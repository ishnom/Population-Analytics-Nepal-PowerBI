# Nepal Census 2021 Population Dashboard

## Overview
This project presents an interactive Power BI dashboard based on Nepal Census 2021 data. The dashboard focuses on population distribution, gender composition, and household insights across districts and local levels.

## Dashboard Features
- District-level slicer for interactive analysis
- Key KPIs including total population, households, families, average household size, average family size, male population, and female population
- Visual analysis of top populated districts and local levels
- Gender-based population comparison across districts

## Key Visuals
- Top 5 most populated local levels
- Top 7 most populated districts
- Top 5 districts by female population
- Dynamic KPIs driven by slicer selection

## Data Description
The dataset includes district-wise and local-level census information such as:
- Total population
- Male and female population
- Total households and families
- Unique identifiers for accurate aggregation

Supporting tables are used for district-level and local-level population analysis.

## Tools Used
- Power BI
- DAX
- Data modeling and aggregation
- Interactive slicers and visuals
